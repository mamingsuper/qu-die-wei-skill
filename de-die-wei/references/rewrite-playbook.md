# 去爹味改写 Playbook

The goal is not to become vague or timid. The goal is to keep useful content while removing hierarchy, control, unsolicited judgment, and emotional invalidation.

## First Pass: Detect

Before rewriting, scan the text across five dimensions:

- 姿态：是否高高在上、默认自己更懂。
- 边界：是否未经邀请就评价、插手、安排。
- 共情：是否忽略或否定对方感受。
- 目的：是否借建议维护自己的权威、价值观或存在感。
- 结果：对方读完是否更可能感到被压迫、被羞辱、被纠正。

## Rewrite Tactics

### 1. 先问再讲

Use when advice was not explicitly requested.

- Before: "你应该马上换工作。"
- After: "你想听一点建议，还是现在更想先把这件事说完？"

### 2. 分享代替指导

Use when the original relies on commands or universal truths.

- Before: "你就按我说的做。"
- After: "我之前遇到类似情况时试过这个办法，对我有用；你可以看看适不适合你。"

### 3. 共情优先

Use when the original jumps into judgment or solution.

- Before: "这有什么好崩溃的？"
- After: "听起来这件事确实消耗很大。"

### 4. 守住边界

Use when the original comments on private life or unrelated choices.

- Before: "你这个年纪就该结婚了。"
- After: "这是你的生活选择，我不替你下判断。"

### 5. 放下说服欲

Use when the original tries to win the conversation.

- Before: "你不用争，我肯定是对的。"
- After: "我们看法可能不一样，我说说我的理解，你保留自己的判断。"

### 6. 降低姿态

Use when seniority, experience, role, or identity is used as pressure.

- Before: "我吃过的盐比你吃过的饭还多。"
- After: "我有过一些相关经历，但你的情况可能不一样。"

### 7. 多倾听少打断

Use when the original fills every silence with correction.

- Before: "你先别说了，我告诉你问题在哪。"
- After: "你先说完，我听一下完整情况。"

## Replacement Patterns

| 爹味表达 | 去爹味替换 |
|---|---|
| 你应该 | 你可以考虑 / 如果你愿意，可以试试 |
| 听我的 | 我提供一个思路，你自己判断 |
| 我这是为你好 | 我担心的是...但决定权在你 |
| 你还年轻 | 你的情况可能和我的经验不同 |
| 这有什么 | 听起来这件事对你影响不小 |
| 你就是 | 我观察到的是...不知道是否准确 |
| 别矫情 | 这个感受是真实的，我们可以先处理它 |

## Output Quality Bar

- Keep the original practical point if it is useful.
- Make the rewrite concrete, not just polite.
- Keep the speaker's voice unless that voice is the problem.
- Avoid adding "亲", excessive apologies, or performative softness.
- If the original has multiple audiences, rewrite for the actual recipient, not for public performance.
