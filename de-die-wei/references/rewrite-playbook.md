# 去爹味改写指南 (Playbook)

去爹味的最终目的不是让表达变得含糊其辞或胆小怕事。其核心在于：保留有用的内容，同时去除阶级感、控制欲、未经请求的评判以及对他人情绪的否定。

## 第一步：先检测

在改写之前，先从五个维度扫描文本：

- **姿态**：是否高高在上、默认自己更懂。
- **边界**：是否未经邀请就评价、插手、安排。
- **共情**：是否忽略或否定对方感受。
- **目的**：是否借建议维护自己的权威、价值观或存在感。
- **结果**：对方读完是否更可能感到被压迫、被羞辱、被纠正。

## 改写策略

### 1. 先问再讲

当对方没有明确寻求建议时使用。

- **改写前**: "你应该马上换工作。"
- **改写后**: "你想听一点建议，还是现在更想先把这件事说完？"

### 2. 分享代替指导

当原文依赖于命令或试图宣扬某种“绝对真理”时使用。

- **改写前**: "你就按我说的做。"
- **改写后**: "我之前遇到类似情况时试过这个办法，对我有用；你可以看看适不适合你。"

### 3. 共情优先

当原文直接跳到评判或给出解决方案时使用。

- **改写前**: "这有什么好崩溃的？"
- **改写后**: "听起来这件事确实消耗很大。"

### 4. 守住边界

当原文评论对方私生活或做出与己无关的选择时使用。

- **改写前**: "你这个年纪就该结婚了。"
- **改写后**: "这是你的生活选择，我不替你下判断。"

### 5. 放下说服欲

当原文试图在对话中“赢”过对方时使用。

- **改写前**: "你不用争，我肯定是对的。"
- **改写后**: "我们看法可能不一样，我说说我的理解，你保留自己的判断。"

### 6. 降低姿态

当原文利用资历、经验、角色或身份施加压力时使用。

- **改写前**: "我吃过的盐比你吃过的饭还多。"
- **改写后**: "我有过一些相关经历，但你的情况可能不一样。"

### 7. 多倾听少打断

当原文试图用纠正来填满每一个沉默时使用。

- **改写前**: "你先别说了，我告诉你问题在哪。"
- **改写后**: "你先说完，我听一下完整情况。"

## 常见替换句式

| 爹味表达 | 去爹味替换 |
|---|---|
| 你应该 | 你可以考虑 / 如果你愿意，可以试试 |
| 听我的 | 我提供一个思路，你自己判断 |
| 我这是为你好 | 我担心的是...但决定权在你 |
| 你还年轻 | 你的情况可能和我的经验不同 |
| 这有什么 | 听起来这件事对你影响不小 |
| 你就是 | 我观察到的是...不知道是否准确 |
| 别矫情 | 这个感受是真实的，我们可以先处理它 |

## 输出质量标准

- 如果原文的观点有实际用处，请保留它。
- 改写要具体落地，而不仅仅是换成礼貌用语。
- 除非原本的语气就是问题的根源，否则尽量保留说话者个人的表达风格。
- 避免添加“亲~”、过度的道歉或表演性质的软弱语气。
- 如果原文有多个受众目标，请针对“真正的接收者”进行改写，而不是为了公共表演。
